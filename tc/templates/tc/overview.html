About Test Planner

Test Planner is a system to allow you to run through a test sequence and record information along the way. The initial target is to run through the ISTA 3A Test Procedure.

This is currently in Alpha and changing frequently. If you use it please remember:

  • There is currently no security so all data you enter is public
  • All data you enter is subject to being deleted as the site evolves
