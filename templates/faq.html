How to checkin from bash script

You can use curl or wget commands to checkin from a shell script. For example, using curl you would use:

curl -L http://watch4.events/checkin/9YZNCBAR/

Note: currently you must use the -L option on curl so that the domain name redirect is performed

How to checkin from python

There are several ways to checkin from python but this is one example:

try: # For Python 3.0 and later from urllib.request import urlopen except ImportError: # Fall back to Python 2's urllib2 from urllib2 import urlopen response = urlopen("http://watch4.events/checkin/9YZNCBAR/") response_text = response.read()

How to checkin from a windows batch file

In windows you can checkin using powershell. This command passes the url as a command line argument to powershell eliminating the need to create a separate powershell script.

One issue is that watch4.events has a URL re-direct to w4e.herokuapp.com which does not work with the WebClient.Download so you must use the w4e.herokuapp.com domain in the checkin URL as shown below.

powershell -Command "& { (new-object System.Net.WebClient).DownloadFile('http://w4e.herokuapp.com /checkin/CA309J0K','c:\windows\temp\dummy.txt') } "
Note there is no line break after the .com, one is shown above just for legibility.
